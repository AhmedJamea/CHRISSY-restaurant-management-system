# Restaurant Management System (RMS)

A full-stack restaurant management system designed to streamline daily operations, manage orders and payments, and improve workflow efficiency.

## Architecture Overview

This project is built using a **Layered Architecture** to ensure a clean separation of concerns, high maintainability, and scalability. By decoupling the core business logic from the infrastructure and API layers, the system remains flexible and easy to test.

* **Presentation Layer (Web API):** Handles HTTP requests and manages the API surface.
* **Service Layer:** Contains the core business logic and cross-cutting concerns.
* **Data Access Layer:** Manages persistence and database interactions using Entity Framework Core.

---

## 🛠 Tech Stack

* **Framework:** .NET 10 / ASP.NET Core Web API
* **Database:** SQL Server (SSMS)
* **ORM:** Entity Framework Core
* **Security:** ASP.NET Core Identity, JWT (JSON Web Tokens), Google OAuth 2.0
